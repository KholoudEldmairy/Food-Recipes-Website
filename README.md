# Food Recipes Website 🥗

A fully responsive web application for exploring diverse food recipes, featuring a clean user interface and intuitive navigation.

## 🌟 Key Features
* **Full Responsiveness:** Optimized for seamless viewing on desktops, tablets, and mobile devices.
* **Interactive UI:** Dynamic recipe presentation using HTML and CSS.
* **Category Navigation:** Structured layout to browse different cuisines and meal types.
* **Modern Design:** Focused on high-quality visuals and user-centric experience.

## 🛠️ Technical Stack
* **HTML5:** For semantic web structure and accessibility.
* **CSS3:** Advanced styling using **Flexbox** and **CSS Grid** for responsive layouts.
* **JavaScript:** Integrated for basic interactivity and page logic.

## 📂 Project Structure
* `main.html`: The core landing page for the application.
* `pro.html` & `greek.html`: Detailed recipe showcases.
* `main.css`: Centralized styling for consistent branding across the site.
